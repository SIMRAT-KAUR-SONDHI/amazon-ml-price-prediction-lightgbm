# Amazon ML Challenge Models

## Overview

This repository contains machine learning models developed for demand forecasting and predictive analytics using advanced gradient boosting techniques. The project demonstrates complete end-to-end regression workflows including data preprocessing, feature engineering, model training, evaluation, and prediction generation using Python and Scikit-learn.

The repository includes implementations of:

- HistGradientBoostingRegressor
- LightGBM Regressor

---

## Models Implemented

### 1. HistGradientBoostingRegressor
A highly efficient gradient boosting implementation from Scikit-learn optimized for large-scale structured datasets with faster training performance and strong predictive accuracy.

### 2. LightGBM Regressor
A high-performance gradient boosting framework developed by Microsoft, designed for scalability, speed, and efficient handling of large datasets and high-dimensional features.

## Evaluation Metrics

The models are evaluated using standard regression metrics:

- **RMSE** — Root Mean Squared Error
- **MAE** — Mean Absolute Error
- **SMAPE** — Symmetric Mean Absolute Percentage Error
- **R² Score** — Coefficient of Determination
