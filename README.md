# Amazon ML Challenge - Price Prediction using LightGBM

This project focuses on predicting product prices using machine learning techniques.  
The pipeline includes data cleaning, feature engineering, model training, and evaluation using SMAPE.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- LightGBM
- Jupyter Notebook

## Key Features
- Cleaned and preprocessed product data
- Performed feature engineering on structured and text-based data
- Trained LightGBM model for price prediction
- Evaluated model performance using SMAPE metric
- Generated prediction output for submission

- `\Light Gbm Amazon ML challenge.ipynb`
